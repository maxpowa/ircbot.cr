# ircbot.cr

IRC Bot library written in crystal, using fast_irc.cr for message parsing

[![crystal-docs.org](https://crystal-docs.org/badge.svg)](https://crystal-docs.org/maxpowa/ircbot.cr)

## Installation

Add it to `shard.yml`

```yaml
dependencies:
  ircbot:
    github: maxpowa/ircbot.cr
```
